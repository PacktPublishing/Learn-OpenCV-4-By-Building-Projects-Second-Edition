# Chapter 09. Learning Object Tracking 

## Requirements

- OpenCV 4.0
- WebCam

## Steps to build

To compile on linux, Mac or windows using Mingw

```
mkdir build
cd build
cmake ..
make
```

## Executables

The following applications are generated.

```
./camshiftTracker  
./coloredObjectTracker  
./farnebackTracker  
./goodFeaturesToTrack  
./harrisCornersTracker  
./lucasKanadeTracker
```
